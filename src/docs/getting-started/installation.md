# Installation

## Requirements

- Node.js 12+
- NPM / Yarn Classic (1.x)

## Installing

You can install Fasteer.js simply by adding the `@fasteerjs/fasteer` package.

<code-group>
<code-block title="Yarn" active>
```bash
yarn add @fasteerjs/fasteer
```
</code-block>

<code-block title="NPM">
```bash
npm i @fasteerjs/fasteer
```
</code-block>
</code-group>
